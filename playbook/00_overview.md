# HackKit Playbook

## What Is This?

HackKit is a complete, repeatable toolkit for planning and executing a GenAI hackathon at any university. It was built from real experience running the #BeTheAgenticBuilder Hackathon at the University of Maryland (April 2026).

## Who Is This For?

Any AWS employee (CSM, SA, AM, or volunteer) who wants to run a similar event at a university. No prior event planning experience required.

## What You Get

1. **This Playbook** (step-by-step guides for every phase)
2. **Templates** (editable files you customize for your event)
3. **Scoring App** (Streamlit web app for live judging)
4. **Checklists** (nothing falls through the cracks)

## Timeline at a Glance

| When | Phase | Key Actions |
|------|-------|-------------|
| 8 weeks out | Planning | Secure venue, budget, university contact |
| 6 weeks out | Recruitment | Flyer, registration form, outreach |
| 4 weeks out | Content | Finalize agenda, prep PartyRock, brief volunteers |
| 2 weeks out | Logistics | Confirm headcount, catering, AV, swag |
| 1 week out | Final Prep | Dry run, volunteer assignments, print materials |
| Day of | Execution | Follow the run-of-show |
| Day after | Follow-up | Survey, social posts, thank-yous |

## How to Use This Playbook

1. Read this overview
2. Go through each numbered section in order
3. Use the templates in `/templates/` (fill in the bracketed fields)
4. Set up the scoring app (instructions in `06_judging.md`)
5. Run your event
6. Capture lessons learned for the next person

## Quick Start Checklist

- [ ] Identify university and faculty contact
- [ ] Get budget approval (estimate: $2,000-5,000 depending on catering/swag)
- [ ] Recruit 4-6 AWS volunteers
- [ ] Book venue (capacity: 60 students, needs WiFi + projector)
- [ ] Set event date (Friday works best, 10am-3pm)
- [ ] Launch registration (cap at 60, use Smartsheet or Google Form)
- [ ] Customize templates for your university
- [ ] Run the event using this playbook

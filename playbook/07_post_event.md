# 07 - Post-Event Follow-Up (Within 1 Week)

## Step-by-Step

### 1. Send Event Survey (Same Day)
- [ ] Share survey link with all attendees
- [ ] Key questions:
  - Overall satisfaction (1-5)
  - What did you learn?
  - What would you improve?
  - Would you attend another AWS event?
  - Can we use your photo/quote for social media?

### 2. Thank-You Emails (Within 48 Hours)
- [ ] Thank university faculty contact
- [ ] Thank AWS volunteers
- [ ] Thank students (include: survey link, AWS learning resources, PartyRock link)

### 3. LinkedIn Post (Within 1 Week)
- [ ] Use template: `templates/linkedin_post_template.md`
- [ ] Include: event photo, tag university, tag AWS volunteers
- [ ] Hashtags: #GenAI #AWS #PartyRock #WorkingBackwards

### 4. Internal Reporting
- [ ] Log as a Tech Activity in AWSentral (if applicable)
- [ ] Metrics to capture:
  - Number of attendees
  - Number of teams
  - Number of working apps built
  - Student satisfaction score

### 5. Archive Materials
- [ ] Save final presentation deck
- [ ] Save scoring results (CSV from app)
- [ ] Save photos and survey responses
- [ ] Update lessons learned (next section)
